# AbastecePro

AbastecePro é uma Progressive Web App (PWA) para gerir abastecimentos, analisar consumo e custos por veículo. Desenvolvida com HTML5, CSS3 e JavaScript puro, com Firebase (Authentication, Firestore, Hosting, Storage) como backend.

## Funcionalidades principais (MVP)
- Registo/Login (Email/Password)
- Gestão de veículos (CRUD)
- Registo de abastecimentos (CRUD) com validações
- Dashboard com KPIs e gráficos (Chart.js)
- Exportação CSV e suporte offline (service worker + IndexedDB)
